Portal do Candidato ENEM

Projeto desenvolvido para a disciplina de Versionamento do curso Técnico em Desenvolvimento de Sistemas.

O sistema simula um Portal do Candidato do ENEM, permitindo o cadastro e login de usuários, acesso a uma área interna, consulta de cursos por área do conhecimento e realização da inscrição para a prova.

Funcionalidades
Cadastro de candidatos
Login com validação de dados
Painel principal do candidato
Consulta de cursos utilizando filtros por área
Inscrição para a prova do ENEM
Validação de formulários com JavaScript
Navegação entre páginas HTML
Interface desenvolvida com HTML, CSS e JavaScript
Tecnologias Utilizadas
HTML5
CSS3
JavaScript
Git e GitHub
